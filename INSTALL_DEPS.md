# Інсталяція залежностей для XML-UA (Windows)

Цей документ описує, як встановити додаткові Python-бібліотеки, які потрібні для повної роботи плагіна XML-UA.

**Потрібні бібліотеки**
- `xmlschema`
- `docxtpl`
- `pymorphy2`
- `pymorphy2-dicts-uk`

**Рекомендований спосіб (через OSGeo4W)**
1. Відкрийте вікно **OSGeo4W Shell** (або `OSGeo4W.bat`) для вашої інсталяції QGIS.
1. Виконайте команду:

```bat
python -m pip install --upgrade xmlschema docxtpl pymorphy2 pymorphy2-dicts-uk
```

1. Перезапустіть QGIS.

**Як знайти OSGeo4W Shell**
- Якщо QGIS встановлений через OSGeo4W: знайдіть ярлик **OSGeo4W Shell** у меню Пуск.
- Якщо QGIS встановлений окремо: запустіть `o4w_env.bat` у каталозі встановлення QGIS (зазвичай `C:\Program Files\QGIS <версія>\bin\o4w_env.bat`).

**Якщо інсталяція не вдається**
- Скопіюйте команду з попередження плагіна і виконайте її вручну у OSGeo4W Shell.
- Переконайтесь, що у вас є доступ до Інтернету і права на встановлення пакетів.
