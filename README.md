# XML UA Plugin

QGIS plugin for creating cadastral XML files in Ukraine.  
Designed to simplify XML creation for both beginners and GIS professionals.

---

## Features

- Generate valid cadastral XML files from GIS layers  
- Detect topological errors in existing XML files  
- Step-by-step guidance for beginners  
- Advanced features for professionals

---

## Installation

1. Download the repository  
2. Load the plugin in QGIS  
3. Follow the online documentation https://krechkivsky.github.io/xml_ua_docs/
